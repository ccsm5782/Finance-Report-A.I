# 한국 증시 지수별 구성종목 심층분석 기반 일일 투자보고서 자동화 시스템

> **Finance Report A.I (2025.06~2025.07))**  

## 📋 프로젝트 개요

한국 주식시장의 일일 투자 의사결정을 지원하기 위한 종합적인 자동화 분석 시스템입니다. pykrx 라이브러리를 중심으로 KOSPI, KOSDAQ, KRX 시장의 모든 지수 데이터를 실시간으로 수집하여 AI 기반 투자 보고서를 자동 생성합니다.

### 🎯 주요 특징

- **📊 포괄적 시장 분석**: 122개+ 한국 증시 지수 일일 등락률 자동 계산
- **🔍 심층 구성종목 분석**: 상위/하위 4개 지수의 구성종목 개별 성과 분석
- **📈 기술적 분석**: 이동평균선, 지지저항선, 거래량, 외국인/기관 매매 현황
- **📰 실시간 뉴스 통합**: 네이버 뉴스 API 기반 관련 뉴스 자동 수집
- **🤖 AI 보고서 생성**: Google Gemini 2.5 Flash 모델 활용 전문 투자 보고서
- **💾 데이터 저장**: 7개 유형의 분석 결과 파일 자동 저장

## 🏗️ 시스템 아키텍처

```
📦 시장 데이터 수집 (pykrx)

├── 일일 등락률 계산
└── 구성종목 분석

📊 기술적 분석
├── 180일간 OHLCV 데이터
├── 이동평균선 (20/60/120일)
├── 지지저항선 분석
└── 거래량 & 외국인/기관 매매

📰 뉴스 수집 (네이버 API)
├── 지수별 관련 뉴스
└── 개별종목 뉴스

🤖 AI 보고서 생성 (Google Gemini)
├── 시장 전반 흐름 요약
├── 상위 지수 모멘텀 분석

```

## 🛠️ 주요 기능 상세

### 📈 지수 분석
- **데이터 소스**: pykrx 라이브러리
- **분석 범위**: KRX, KOSPI, KOSDAQ 전체 지수
- **분석 지표**: 일일 등락률, 성과 랭킹

### 🔍 구성종목 분석
- **선별 기준**: 등락률 상위 2개, 하위 2개 지수
- **분석 항목**: 개별종목 등락률, 지수 대비 상대 성과
- **기술적 지표**: 
  - 이동평균선 (20일/60일/120일)
  - 지지저항선 (120일 고점, 20일 저점)
  - 거래량 변화 (전일 대비, 20일 평균 대비)
  - 외국인/기관 5일간 순매매

### 📰 뉴스 수집
- **API**: 네이버 뉴스 검색 API
- **필터링**: 발행일자 기준, 중복 제거
- **키워드**: 지수명, 종목명 + "주식/주가"

### 🤖 AI 보고서
- **모델**: Google Gemini 2.5 Flash
- **구조**: 3단계 분석 (시장 전반 → 상위 지수 → 하위 지수)
- **특징**: 전문적인 금융 분석가 톤, 구체적인 투자 전략 제시

## 📚 API 키 발급 가이드

### Google Gemini API
1. [Google AI Studio](https://aistudio.google.com/app/apikey) 접속
2. "Create API Key" 클릭
3. 생성된 키를 `.env` 파일에 입력

### 네이버 뉴스 API
1. [네이버 개발자센터](https://developers.naver.com/) 접속  
2. "Application 등록" → "뉴스" 선택
3. Client ID와 Client Secret를 `.env` 파일에 입력


## 🚀 추후 보완 및 발전 방향

본 프로젝트는 현재 실시간 데이터 수집 및 AI 보고서 생성을 기본으로 하며, 향후 다음과 같은 고도화를 계획하고 있습니다.

### 1. 지식 베이스 구축 (RAG 고도화)
* **벡터 데이터베이스(Vector DB) 도입**: 과거의 시장 분석 보고서 및 뉴스 데이터를 벡터화하여 저장함으로써, 현재 시장 상황과 유사한 과거 사례를 정교하게 검색하고 분석에 활용합니다.
* **Elasticsearch 적재**: 대량의 텍스트 데이터를 엘라스틱서치에 적재하여 키워드 기반의 고속 검색과 복합 쿼리를 지원하며, 분석 리포트의 히스토리 관리 효율성을 높입니다.

### 2. 하이브리드 검색 시스템
* 엘라스틱서치의 키워드 검색(BM25)과 벡터 DB의 의미론적 검색(Semantic Search)을 결합한 하이브리드 검색을 통해 AI 보고서의 근거 자료 확보 능력을 극대화할 예정입니다.
